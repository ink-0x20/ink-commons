## 概要
Javaの外部ライブラリを使用せずに作成したライブラリです。

会社などで外部ライブラリを導入できない場合や、Javaの勉強用に処理を知りたい場合など、お役に立つと幸いです。

昨今外部ライブラリで脆弱性が見つかるなど、不安な方も是非参考程度に・・・。

## 開発環境
Java17

- 各種パッケージ
